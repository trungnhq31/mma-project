# EVCare Backend (BE-se183262)

## Setup

1. `cp .env.example .env`
2. `docker-compose up -d`
3. `npm run dev`

## APIs

- GET `/api/v1/vehicle-type`
- GET `/api/v1/appointment/service-mode`
- GET `/api/v1/service-type/vehicle_type/:id`
- POST `/api/v1/appointment`
